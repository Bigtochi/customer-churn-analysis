# Customer-churn-analysis

# Project Overview
This project focused on analyzing customer behavior data to understand the factors that may contribute to customer churn. Using exploratory data analysis and visualizations, I examined customer patterns and behaviors to identify trends associated with customers leaving a service.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Findings
- Customers who churned had a higher number of support calls compared to retained customers.
- Payment delays appeared to have a relationship with customer churn.
- Usage frequency differed between retained and churned customers.
- Customer tenure showed noticeable differences in customer retention.

## Business Recommendations
- Improve customer support processes and response quality.
- Reduce payment-related issues.
- Monitor low-activity customers earlier.
- Develop retention strategies for customers at higher risk of churn.

## Conclusion
This analysis shows that customer behavior can provide useful insights into customer retention patterns and help businesses make better decisions.
